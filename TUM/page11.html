Single Transformer Backbone

  • Uses plain Vision Transformer (e.g., DINOv2) without architectural specialization.

  • Total $L$ blocks divided into two groups: $L_s$ (local) and $L_g$ (global).

  • Empirical optimal ratio found at $L_s : L_g = 2 : 1$.

  • Inherits scaling properties directly from large-scale pretraining.

Input-Adaptive Cross-View Attention

  • First $L_s$ layers: standard self-attention within each image (independent).

  • Next $L_g$ layers: alternate between cross-view and within-view via token reordering.

  • Adaptive: If $N_v=1$, naturally reduces to monocular depth estimation without overhead.

  • Scalable: No extra 3D cost volumes; scales linearly with total token count.
